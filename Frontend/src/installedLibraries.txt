Json Server zum Testen von Backend-Anbindung ohne Backend
npm install --save-dev json-server

{
  "scripts": {
    "api": "json-server --watch db.json --port 3001"
  }
}
im Package.json damit dann mit " npm run api " in der Konsole der Json Server gestartet werden kann. WIEDER ENFERNEN WENN JSON SERVER WEGKOMMT!
